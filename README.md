# Econometrics-I Term Paper
This is the repository for a term/technical paper written as part of my econometrics course


## Requisites for replication
The replication requires the raw .dta files from IHDS-2. These files are too large to be included in the GitHub repo, but here are the instructions for replication:

1. Download the files from here: https://www.icpsr.umich.edu/web/DSDR/studies/36151
2. From DS0001 to DS0003 subfolders, copy the 36151-0001-Data.dta to 36151-0003-Data.dta files to the 02_raw subfolder

Once this is done, the STATA .do files and the EDA/graph Python script should run just fine


## Output (graphs, tables, etc)
Once the .do files are executed and the notebook has been run, the folder will be populated with .tex tables as well as the graphs used in the term paper.
